# AgriTech Farmora Backend

FastAPI Boilerplate.

## Setup

1.  Create a virtual environment (already created locally):
    ```bash
    python -m venv venv
    ```

2.  Activate the virtual environment:
    -   Windows: `.\venv\Scripts\activate`
    -   Linux/Mac: `source venv/bin/activate`

3.  Install dependencies (already installed locally):
    ```bash
    pip install -r requirements.txt
    ```

## Run

```bash
uvicorn main:app --reload
```
